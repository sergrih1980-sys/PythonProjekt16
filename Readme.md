# Проект: Анализ вакансий с hh.ru


Приложение для сбора данных о вакансиях и работодателях с hh.ru и их анализа через PostgreSQL.

## Функциональность


Проект позволяет:
* получать данные о работодателях по их ID через API hh.ru;
* собирать вакансии для конкретных работодателей;
* хранить данные в PostgreSQL;
* выполнять аналитические запросы:
  * список компаний и количество вакансий у каждой;
  * все вакансии с указанием компании, зарплаты и ссылки;
  * средняя зарплата по вакансиям;
  * вакансии с зарплатой выше средней;
  * поиск вакансий по ключевому слову в названии.

## Технологии

* **Python 3.x** — основной язык программирования;
* **psycopg2** — для взаимодействия с PostgreSQL;
* **requests** — для запросов к API hh.ru;
* **PostgreSQL** — СУБД для хранения данных;
* **python-dotenv** — для работы с переменными окружения.
